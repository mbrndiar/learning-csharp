# 🧭 Lessons

The 15 lessons form the prerequisite-safe path through modern C# and .NET.
Complete runnable demonstrations live with the lesson that explains them; they
are not learner TODO files. Active implementation work lives separately under
[`exercises/`](../exercises/README.md).

## 🔁 Study loop

1. Read the lesson objectives and prerequisites.
2. Predict a runnable program's output or failure.
3. Run it from the repository root.
4. Change one input or condition and explain the result.
5. Complete the matching exercise before reading its solution.
6. Answer the review questions without looking back.

## 🗺️ Course order

| Lesson | Main outcome |
| --- | --- |
| [01 · First program](01_first_program/) | Run a file-based C# app and use the predict-run-modify loop. |
| [02 · Values, types, and null](02_values_types_and_null/) | Choose representations, conversions, nullable values, and checked arithmetic. |
| [03 · Decisions and repetition](03_decisions_and_repetition/) | Express branches and trace repeated state changes. |
| [04 · Collections and iteration](04_collections_and_iteration/) | Choose containers and reason about iteration, aliases, copies, and equality. |
| [05 · Methods, errors, and debugging](05_methods_errors_and_debugging/) | Decompose behavior, predict parameter effects, and diagnose failures. |
| [06 · Projects, solutions, and builds](06_projects_solutions_and_builds/) | Use normal SDK projects, references, packages, lock files, and build output. |
| [07 · Modeling data and behavior](07_modeling_data_and_behavior/) | Protect invariants and distinguish dates, instants, durations, and clocks. |
| [08 · Abstractions, generics, and delegates](08_abstractions_generics_and_delegates/) | Reuse and substitute behavior through explicit typed contracts. |
| [09 · LINQ and transformations](09_linq_and_transformations/) | Build deliberate sequence pipelines and reason about their cost. |
| [10 · Testing and dependency boundaries](10_testing_and_dependency_boundaries/) | Write behavioral tests and control volatile dependencies. |
| [11 · Files, streams, and JSON](11_files_streams_and_json/) | Cross representation and file-I/O boundaries safely. |
| [12 · Async, cancellation, and concurrency](12_async_cancellation_and_concurrency/) | Own asynchronous and concurrent work explicitly. |
| [13 · SQL and SQLite](13_sql_and_sqlite/) | Model relational data and use SQLite through parameterized, transactional ADO.NET. |
| [14 · HTTP clients and Minimal APIs](14_http_clients_and_minimal_apis/) | Understand HTTP mechanics, middleware, Minimal APIs, and raw/typed clients. |
| [15 · Application composition](15_application_composition/) | Compose selectable adapters and run the complete quality loop. |

After lesson 15, complete the required [Tasks applied
project](../projects/tasks/README.md), then both [capstone
tracks](../capstones/README.md).
