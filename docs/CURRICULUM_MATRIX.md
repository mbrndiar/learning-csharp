# Curriculum evidence matrix

This matrix is the durable course-level conformance record. A row becomes
`covered` only after its learner-facing explanation, runnable demonstration,
practice feedback, and any required independent application have been
verified. Topic names and incidental use do not count as evidence.

The final reconciliation used `course-manifest.json`, runnable samples,
solution-selected MTP tests, starter-feedback checks, the reading-log acceptance
suite, and the root Release workflow. The recorded validation baseline is 14
units, 19 independently executed samples, 188 passing solution tests, 65
passing capstone tests, and 87.1% capstone branch coverage.
