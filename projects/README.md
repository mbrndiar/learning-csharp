# 🧩 Applied projects

Applied projects combine several lessons behind one observable contract. They
are required course work, but smaller and more guided than final capstones.

## ✅ Required project

- [`tasks/`](tasks/README.md) builds one Task domain through SQLite and
  versioned Markdown repositories, low-level ASP.NET Core middleware and Minimal
  API servers, raw and typed HttpClient transports, and one CLI policy.

Complete its five milestones after lesson 15 and before either capstone. The
starter and solution expose matching public boundaries, while shared tests
verify repository, HTTP, client, process, and interoperability behavior.
