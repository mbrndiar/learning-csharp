# 🏆 Capstones

Complete both capstones after the required [Tasks applied
project](../projects/tasks/README.md). Each track has a compileable guided
starter, a complete reference solution, shared milestone tests, and an
independent coverage gate.

## 🔁 Comparative track

[`comparative/`](comparative/README.md) implements the frozen cross-language
`comparative-kv` 1.0.0 contract: an exact SQLite-backed versioned key/value CLI
with restricted JSON, migration, revisions, compare-and-set behavior, and real
multi-process contention.

## 💎 Idiomatic C# track

[`idiomatic/`](idiomatic/README.md) is the Reading Log application: a C#-focused
domain/application/infrastructure design with asynchronous JSON persistence,
Minimal API, typed HttpClient CLI, cancellation, strict dates, and deterministic
integration tests.

## 🧭 Recommended order

1. Finish every lesson and its exercise.
2. Complete `projects/tasks/`.
3. Build `capstones/comparative/` against the frozen contract.
4. Build `capstones/idiomatic/` as the open C#-specific synthesis.
